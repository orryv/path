## Files and Directories usage

`Path::absolute()` must accept:
  - Windows drive paths: `C:/projects/demo/file.txt`
  - Unix/Posix absolute paths: `/projects/demo/file.txt`
  - Windows UNC paths: `//server/share/folder/file.txt`
  - Windows Extended-length paths: `\\?\C:\projects\demo\file.txt` or `\\?\UNC\server\share\folder\file.txt`
  - URIs: `file:///C:/...`, `http://...`, `ftp://...`, ...

```php
// TODO: check that we are correct that there is no way to figure out if a path is a file or directory when we don't every call the system?

// All methods are immutable, meaning they return new instances and do not modify the current instance.
use Orryv\Path;

# Factory (returns Path instance)
$path = Path::new('C:/projects/demo/file.txt'); // only absolute paths allowed
// OR:
// $path = Path::dir('C:/projects/demo/'); // trailing slash optional
// $path = Path::file('C:/projects/demo/file.txt');
// $path = Path::dot('C:/projects/demo/file.txt'); // will detect if file or directory based on trailing dot or slash
// $path = Path::system('C:/projects/demo/file.txt'); // will detect if file or directory based on system (URI that is not file will throw exception)

# Path resolving (returns FilePath or DirectoryPath instance)
$path = $path->asFile();          // Path as a file (will internally strip last part of the path)
$path = $path->asDirectory();     // Path as a directory
$path = $path->asResolve($detection_method = 'dot'); // other methods could be 'system' (URI that is not file will throw exception), 'dot' means it will look for a . at the end of the path
// or resolveAs($type = 'file'|'directory'|'dot'|'system') ?

# Retrieval (returns AccessPath, ReferencePath or AccessURI instance)
$path->getAccessPath();      // C:\projects\demo\file.txt
$path->getReferencePath();  // C:/projects/demo/file.txt
$path->getAccessURI();      // file:///C:/projects/demo/file.txt
// Windows specific
$path->getAccessPath()->toWindowsLongPath();        // If longer than max path length will return "\\?\C:\projects\demo\file.txt", otherwise same as getAccessPath()

# Options (called on Path, FilePath, DirectoryPath, AccessPath, ReferencePath or AccessURI instances)
$path->preserveEndSlash($value = true); // true means directory paths will always end with a slash, false means they never will (except for root directories). default is true.

# Navigation
$path->setBasePath(DirectoryPath|string Path::new('C:/projects/')->asDirectory()); // Returns DirectoryPath instance Set base path for relative paths. if string will internally convert to DirectoryPath instance
$path->cd('..'); // Returns DirectoryPath instance

# AccessPath, ReferencePath or AccessURI methods
$path->toString();          // C:/projects/demo/
```
